# Charades Game Application

## Overview

This project presents a digital rendition of the classic party game, Charades.
Developed as part of a coursework assignment, the application allows users to engage in an interactive and entertaining game
where players act out words or phrases for others to guess. The project emphasizes user-friendly design and real-time
responsiveness to provide an engaging user experience.

## Key Features

- **Interactive Gameplay**: Enables users to play Charades digitally, facilitating both single and multiplayer modes.
- **User-Friendly Interface**: Designed with simplicity and ease of use in mind, ensuring accessibility for users of all ages.
- **Real-Time Updates**: Incorporates real-time feedback and updates to enhance the gaming experience.

## Project Structure

- `main.py`: The main script that initializes and runs the application.
- `game_logic.py`: Contains the core logic and rules governing the gameplay.
- `ui_components/`: Directory housing the user interface components and layouts.
- `assets/`: Includes images, icons, and other media assets used in the application.

## Getting Started

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/sinchan-hegde/F24_project3_charades.git
   ```

2. **Navigate to the Project Directory**:
   ```bash
   cd F24_project3_charades
   ```

3. **Install Dependencies**:
   Ensure you have the required packages installed. You can install them using:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the Application**:
   ```bash
   python main.py
   ```
